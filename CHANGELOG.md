# Castle.Transactions (Castle.Services.Transaction &amp; Castle.Facilities.AutoTx) - Changelog

## 5.3.x (202x-xx-xx)

### All

Breaking Changes:

- Replaced Castle.Services.Transaction.IsolationMode with System.Transactions.IsolationLevel.
